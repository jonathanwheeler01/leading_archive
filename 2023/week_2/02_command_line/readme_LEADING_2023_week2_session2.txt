# LEADING Boot Camp 2023, Week 2 Session 2

**Command Line and Scripting in the Data Science Pipeline**

This directory contains files and content related to the 2023 LEADING boot camp session on command line usage. Topics addressed include 

* Shell and command lines (for Linux, Mac, and Windows).
* Piping and redirection, power to combine tools.

The session includes a hands-on case study and demonstration. The session includes two videos, one for the lecture and one for the hands-on demonstration. Links are provided below, in the *Resources* section.

## Included files

1. cmd_exercise.pdf
	- Commands executed for the hands-on demonstration. Note the demonstration uses a dataset downloaded from a repository by GitHub user tommyhegarty, who cites kaggle user LiamLarsen as the origina data source. Both repostories are cited below.
2. cmd_ke.pdf
	- Slide deck for the session. Overview of basic command line tools for data management and text processing, how to navigate directories, manipulate files, and process text. Includes an introduction to regular expressions.
