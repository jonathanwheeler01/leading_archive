# LEADING Boot Camp 2023, Week 3 Session 2

**Graph/Network Models and Matrix Processing**

This directory contains files and content related to the 2023 LEADING boot camp session on graph and network models and matrix processing. Topics addressed include 

* What are graph/network models?
* Sources of graph/network data and an introduction into types of models.
* Tools used for representing and analyzing graph/network data.


## Included files

1. 11-LEADING-BootCamp-connectivity.html
	- HTML rendering of the Jupyter Notebook for the session.
2. connectivity.zip
	- Zip file of session materials.
3. 11-LEADING-BootCamp-connectivity.ipynb
	- Included in the zip file. Jupyter Notebook for the session. Introduces concepts related to graph theory, including nodes and edges, and discusses how to represent networks computationally through structures like adjacency matrices, using examples and code implementations for working with network data.
4. newstweet-subsample-linked.json
	- Included in the zip file, in the 'data' directory. Dataset used for extended network analysis example in the Jupyter Notebook. Sampled from the NewsTweet dataset, cited below.
5. twitter_combined.txt
	- Included in the zip file, in the 'data' directory. Dataset used for network visualization example in the Jupyter Notebook. Available from the Stanford Network Analysis Project (SNAP), cited below.
6. random-graph-walk.jpg
	- Included in the zip file, in the 'img' directory. Resource used in the Jupyter Notebook.
