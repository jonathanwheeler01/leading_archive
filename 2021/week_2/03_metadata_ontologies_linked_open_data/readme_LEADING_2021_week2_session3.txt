# LEADING Boot Camp 2021, Week 2 Session 3

**Big Metadata, Ontologies, and Linked Open Data**

This directory contains files and content related to the 2021 LEADING boot camp session on metadata. Topics addressed include:

* Metadata to big metadata.
* Ontologies and Protege.
* Linked open data.

## Included files

1. LEADING.pdf
	- PDF of the session slides.
2. LEADING.pptx
	- Session slides in source MS Powerpoint format. Included to preserve animations that are not included in the PDF copy. Topics covered include ontologies, knowledge representation, and linked data.
3. LEADING-ProtegeTutorial.pdf
	- Tutorial for using the Protégé tool at https://webprotege.stanford.edu/.
4. LD-LAB-W3C-Validator.pdf
	- Lab guide for creating and validating RDF using the W3C RDF Validator.
5. RDF-EX1.txt
	- Text file used in LD-LAB-W3C-Validator.pdf lab guide and demonstration.
