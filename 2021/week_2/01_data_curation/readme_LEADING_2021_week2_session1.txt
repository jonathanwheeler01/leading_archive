# LEADING Boot Camp 2021, Week 2 Session 1

**Data Curation**

This directory contains files and content related to the 2021 LEADING boot camp session on data curation. Topics addressed include 

* The data lifecycle.
* Data curation principles.
* Data curation practices.

## Included files

1. LEADING-Data Curation.pdf
	- PDF of the session slides. Covered topics include data, big data, the curation lifecycle, metadata, and sharing.
