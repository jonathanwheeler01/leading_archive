# LEADING Boot Camp 2021, Week 1 Session 3

**Data Management: SQL and NoSQL**

This directory contains files and content related to the 2021 LEADING boot camp session on data management using SQL and NoSQL applications. Topics addressed include 

* Understand SQL as a database language.
* Use SQL for data science.
* Understand differences between SQL and NoSQL.

## Included files

1. LEADING-NoSQL-2021.pdf
	- Slides for the NoSQL part of the session. Covers the differences between SQL and NoSQL databases, NoSQL models including key-value stores, column stores, and document stores. Includes a MongDB tutorial.
2. LEADING-SQL-2021.pdf
	- Slides for the session. Covers the relational database model, the use of SQL for data engineering and analytics, and basic SQL commands such as joins, subqueries, and aggregation functions.
