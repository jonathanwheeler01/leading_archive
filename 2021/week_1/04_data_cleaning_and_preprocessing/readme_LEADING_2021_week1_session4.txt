# LEADING Boot Camp 2021, Week 1 Session 4

**Data Cleaning and Preprocessing**

This directory contains files and content related to the 2023 LEADING boot camp session on data cleaning. Topics addressed include 

* Dealing with messy data.
* Creating a data cleaning pipeline.
* Data transformation.

## Included files

1. Week 1-Thursday-session1 Data Cleaning and Proprocessing_wExerciseKey.html
	- HTML rendering of the Jupyter Notebook used for the session.
2. Week1_DataCleaning_Proprocessing.zip
	- Zip file of session materials.
3. Week 1-Thursday-session1 Data Cleaning and Proprocessing_wExerciseKey.ipynb
	- Included in the zip file. Jupyter Notebook titled "Data Cleaning and Preprocessing." Demonstration of a data cleaning and preprocessing workflow, with explanations of relevant concepts.
4. housing.csv
	- Included in the zip file, in the 'data' directory. The dataset is available through the Python *scikit-learn* library and from https://lib.stat.cmu.edu/datasets/. Please see dataset citation, below.
