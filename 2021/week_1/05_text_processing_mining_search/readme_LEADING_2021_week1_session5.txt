# LEADING Boot Camp 2021, Week 1 Session 5

**Text Processing, Mining, and Search**

This directory contains files and content related to the 2021 LEADING boot camp session on text processing, mining, and search. Topics addressed include 

* Normalization, stop words, stemming, n-grams.
* Information retrieval (IR) methods.
* Similarity evaluation

## Included files

1. LEADINGbootcamp-slides-textprocessing.pdf
	- Slide deck for the session.
2. ngrams.html
	- HTML rendering of the ngrams Jupyter Notebook. 
3. ngrams.ipynb
	- Jupyter Notebook demonstration of n-grams.
4. regex.html
	- HTML rendering of the regex Jupyter Notebook.
5. regex.ipynb
	- Jupyter Notebook demonstration of regular expressions.
6. stem_example.html
	- HTML rendering of the stem_example Jupyter Notebook.
7. stem_example.ipynb
	- Jupyter Notebook demonstration of stemming.
