# LEADING Boot Camp 2021, Week 1 Session 2

**Data Science Overview**

This directory contains files and content related to the 2021 LEADING boot camp session on data science. Topics addressed include 

* What is data science and who does it?
* Data science life cycles.
* Introduction to common tools used in data science.

## Included files

1. 01-LEADING-BootCamp-introduction(1).html
	- An HTML rendering of the Jupyter Notebook used for the session.
2. introduction.zip
	- A zip file of session materials.
3. 01-LEADING-BootCamp-introduction.ipynb
	- Included in the introduction.zip. A Jupyter Notebook entitled "Chapter 1: Data science and scientific paradigms." Includes discussion of data science concepts and analogical reasoning.
4. google_books-fs.npy
	- Included in the introduction.zip file, in the 'data' directory. A Python array file.
5. google_books-ranks.json
	- Included in the introduction.zip file, in the 'data' directory. A JSON file containing key:values pairs of word ranks from Google Books.
6. msr.csv
	- Included in the introduction.zip file, in the 'data' directory. Syntactic test set of analogy questions, described in Mikolov, Yih, and Zweig (2013). Formerly available online at http://research.microsoft.com/en-us/projects/rnn/default.aspx.
