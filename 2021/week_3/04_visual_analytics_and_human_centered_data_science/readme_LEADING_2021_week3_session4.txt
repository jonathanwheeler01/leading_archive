# LEADING Boot Camp 2021, Week 3 Session 4

**Visual Analytics and Human Centered Data Science**

This directory contains files and content related to the 2021 LEADING boot camp session on visual analytics and human centered data science (HCDS). Topics addressed include 

* Fundamentals of visual analytics.
* Misuse of visualizations.
* Best practice of visualizing data.


## Included files

1. BootCamp-InfoViz.pdf
	- Session slides. Overview of key concepts in data visualization, including mental models, misleading visualizations, and data type taxonomies. See below for citations and links to datasets used to demonstrate Gephi (Beveridge and Hunger, nd.) and Tableau ("Earthquakes 1900-2013") in the session's recorded Zoom video (not included in the public archive).
2. Human-Centered Data Science-final.pdf
	- HCDS slide deck. Topics covered include integration of human-centered principles into data science, data ethics, design, and privacy.
