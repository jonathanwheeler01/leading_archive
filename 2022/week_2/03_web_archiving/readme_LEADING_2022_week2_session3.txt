# LEADING Boot Camp 2022, Week 2 Session 3

**Data Preservation and Web Archiving**

This directory contains files and content related to the 2022 LEADING boot camp session on data preservation and web archiving. Topics addressed include 

* Archival crawling.
* Access and replay.
* Formats and metadata.

## Included files

1. LEADING (2022) - Web Archiving.pdf
	- Slide deck for the session. Topics covered include issues and strategies for archiving the web, including archival crawling, access, replay, metadata formats, and tools like Internet Archive, Archive-It, and WebRecorder.
