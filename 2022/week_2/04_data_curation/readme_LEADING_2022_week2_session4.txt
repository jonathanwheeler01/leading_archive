# LEADING Boot Camp 2022, Week 2 Session 4

**Data Curation**

This directory contains files and content related to the 2022 LEADING boot camp session on data curation. Topics addressed include 

* The data lifecycle.
* Data curation principles.
* Data curation practices.

## Included files

1. LEADING-Data Curation-2022.pdf
	- PDF of the session slides. Covered topics include data, big data, the curation lifecycle, metadata, and sharing.
